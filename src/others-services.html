Организационное сопровождение долговых программ

АО «НФК-СИ» оказывает услуги организационно-технического характера, связанные с размещением облигаций материнской компании ООО «Пионер-Лизинг», в том числе является участником в опционных программах, базовым активом по которым выступают облигации материнской компании

На текущий момент АО «НФК - СИ» предлагает
инвесторам 3 опционные программы: *

Предложение внебиржевых опционов PUT для 2-го выпуска биржевых облигаций ООО «Пионер-Лизинг» (ПионЛизБП2)

Предложение внебиржевых опционов PUT для 3-го выпуска биржевых облигаций ООО «Пионер-Лизинг» (ПионЛизБП3)

Предложение внебиржевых опционов PUT для выпуска облигаций, предназначенных для квалифицированных инвесторов (ПионЛизБР5)

* Только для квалифицированных инвесторов. Не является индивидуальной инвестиционной рекомендацией. Сделка с опционом Put проводится с участием АО "НФК-Сбережения" - лицензия на осуществления брокерской деятельности № 097-10053-100000 без ограничения срока действия.

Поручителем по обязательствам в рамках опционных программ является ООО «Пионер-Лизинг». Внебиржевые опционы PUT доступны для клиентов брокера АО «НФК-Сбережения» через сервис личного кабинета.
